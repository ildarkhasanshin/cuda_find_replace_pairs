Plugin for CudaText
Simple plugin for add/get pairs of find/replace.

Plugin gives few commands in the menu "Plugins / Find-Replace-Pairs":
"List" - show menu-dialog with list of pairs find/replace (* - regex)
"Add" - fill in the find/replace-fields before calling this command
"Remove" - delete pair of find/replace which you created before
